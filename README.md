## вңЁ What is AutoMax?

**AutoMax** is a full-stack Django web application that brings buyers and sellers together in a sleek, modern car marketplace. Whether you're listing your first car or hunting for your dream ride вҖ” AutoMax makes it seamless.

## вҡҷпёҸ Getting Started

### Prerequisites
- Python 3.10+
- pip
- Git

### 1пёҸвғЈ Clone the Repository

```bash
git clone https://github.com/Sukhjas1314/AutoMax-django_project.git
cd AutoMax-django_project
```

### 2пёҸвғЈ Create & Activate Virtual Environment

```bash
# Create the virtual environment
python -m venv venv

# Activate it
source venv/bin/activate      # рҹҗ§ Mac / Linux
venv\Scripts\activate         # рҹӘҹ Windows
```

### 3пёҸвғЈ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4пёҸвғЈ Apply Migrations

```bash
python manage.py migrate
```

### 5пёҸвғЈ (Optional) Create Superuser

```bash
python manage.py createsuperuser
```

### 6пёҸвғЈ Run the Development Server

```bash
python manage.py runserver
```

> рҹҢҗ Open your browser and visit: **http://127.0.0.1:8000**
