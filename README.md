# 🚗 AutoMax - Django Car Marketplace

> A full-stack Django web application for buying and selling cars with modern UI, authentication, and dynamic listings.

---

## 🌟 Features

✨ User Authentication (Register / Login / Logout)
✨ Car Listings with Images & Detailed Information
✨ Advanced Search & Filtering System
✨ Seller Dashboard for Managing Listings
✨ Contact Seller Functionality
✨ Admin Panel for Complete Control
✨ Fully Responsive UI

---

## 🛠️ Tech Stack

* **Backend:** Django, Python
* **Frontend:** HTML, CSS, JavaScript, Bootstrap
* **Database:** SQLite (can be upgraded to PostgreSQL)
* **Version Control:** Git & GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Sukhjas1314/AutoMax-django_project.git
cd AutoMax-django_project
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
source venv/bin/activate   # Mac/Linux
venv\Scripts\activate      # Windows
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Apply Migrations

```bash
python manage.py migrate
```

### 5️⃣ Run Development Server

```bash
python manage.py runserver
```
